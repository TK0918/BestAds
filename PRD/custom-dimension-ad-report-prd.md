## 2. 需求背景&目标

### 2.1 需求背景

- 当前广告客户表现报表已有客户维度，广告账户维度，Campaign维度，Adset维度和Ad维度等固定 Tab.
- 现有报表主要按天输出数据，当业务需要对比两个客户在某个时间段内的总消耗时，需要先导出明细，再自行汇总.
- 业务希望在现有广告客户表现页面新增 `自定义` Tab，通过自定义统计维度直接得到汇总结果.
- 业务希望筛选字段贴近真实投放排查场景，支持按商户，客户名称，负责人，平台，账户，Campaign，Adset，Ad，国家，行业和状态等条件定位数据.
- 业务希望页面展示结果就是汇总后的结果，导出结果也与页面保持一致.

### 2.2 需求目标

- 在 `广告客户表现` 页面新增 `自定义` Tab，底层数据最细粒度与现有 Ad 层数据保持一致.
- 支持业务自定义是否使用日期，客户，投放平台，账户类型，广告账户，Campaign，Adset，Ad，国家，行业作为统计维度.
- 支持根据已选维度动态展示列表字段.
- 支持按已选维度聚合指标，未选择的维度不展示字段，指标向上汇总.
- 支持业务在页面直接完成多客户某时间段消耗对比，不再依赖导出后手工汇总.

### 2.3 核心规则

| 规则 | 说明 |
| --- | --- |
| 底层粒度 | 使用 Ad 层数据作为最细粒度，即可向上汇总到任意已支持维度组合 |
| 维度选择 | 用户勾选的维度作为列表分组字段 |
| 列表展示 | 用户勾选什么维度，列表展示该维度对应字段 |
| 指标汇总 | 未勾选的维度不展示，对应数据会被聚合到已选维度上 |
| 客户字段 | 未勾选客户维度时，列表不展示 `商户ID` 和 `客户名称` |
| 素材字段 | 未勾选 Ad 维度时，列表不展示 `素材预览` 和 `URL` |
| 名称聚合 | Campaign，Adset，Ad 等对象统计时必须使用 ID 聚合，名称仅用于展示 |
| 导出规则 | 导出字段，排序，指标口径与当前页面展示保持一致 |

## 4. 筛选字段

筛选字段只用于限定数据范围，不代表一定会在列表中展示. 列表是否展示某个字段，取决于自定义维度是否勾选对应维度.

| 序号 | 筛选字段 | 控件类型 | 默认值 | 是否必填 | 字段说明 |
| --- | --- | --- | --- | --- | --- |
| 1 | 时间范围 | 日期范围选择器 | 最近7天 | 是 | 统计数据的起止日期 |
| 2 | 商户ID | 文本 | 空 | 否 | 支持多个商户ID筛选，用于精确定位客户 |
| 3 | 客户名称 | 文本 | 空 | 否 | 支持模糊搜索 |
| 4 | BD | 下拉多选 | 空 | 否 | 按商务负责人筛选 |
| 5 | AM | 下拉多选 | 空 | 否 | 按运营负责人筛选 |
| 6 | 投放平台 | 下拉多选 | 空 | 否 | 按媒体或投放平台筛选，例如 Facebook，Google，TikTok |
| 7 | 广告账户ID | 文本 | 空 | 否 | 支持模糊搜索 |
| 8 | 广告账户名称 | 文本 | 空 | 否 | 支持模糊搜索 |
| 9 | 账户类型 | 下拉多选 | 空 | 否 | 按账户类型筛选，例如普通户，绑卡户，代理户 |
| 10 | Campaign ID | 文本 | 空 | 否 | 支持模糊搜索 |
| 11 | Campaign名称 | 文本 | 空 | 否 | 支持模糊搜索 |
| 12 | Adset ID | 文本 | 空 | 否 | 支持模糊搜索 |
| 13 | Adset名称 | 文本 | 空 | 否 | 支持模糊搜索 |
| 14 | Ad ID | 文本 | 空 | 否 | 支持模糊搜索 |
| 15 | Ad名称 | 文本 | 空 | 否 | 支持模糊搜索 |
| 16 | 国家 | 下拉多选 | 空 | 否 | 按国家或地区筛选 |
| 17 | 行业 | 下拉多选 | 空 | 否 | 按客户所属行业筛选 |
| 18 | Campaign状态 | 下拉多选 | 空 | 否 | 按 Campaign 当前状态筛选 |
| 19 | Adset状态 | 下拉多选 | 空 | 否 | 按 Adset 当前状态筛选 |
| 20 | Ad状态 | 下拉多选 | 空 | 否 | 按 Ad 当前状态筛选 |

示例:

| 操作 | 结果 |
| --- | --- |
| 筛选客户A和客户B，只勾选客户维度 | 列表展示客户A和客户B各自的汇总数据 |
| 筛选客户A和客户B，不勾选客户维度，只勾选国家维度 | 列表按国家汇总客户A和客户B的数据，不展示商户ID和客户名称 |
| 筛选客户A，勾选Campaign维度 | 列表展示客户A范围内的 Campaign 汇总数据，不展示商户ID和客户名称，除非同时勾选客户维度 |

## 7. 维度选择后的列表展示

### 7.1 常见维度组合

| 勾选维度 | 列表展示维度字段 | 指标统计方式 | 适用场景 |
| --- | --- | --- | --- |
| 客户 | 商户ID，客户名称 | 按客户汇总整个时间范围数据 | 对比多个客户在某时间段的总消耗 |
| 日期 + 客户 | 日期，商户ID，客户名称 | 按日期+客户汇总 | 对比多个客户每日消耗趋势 |
| 投放平台 | 投放平台 | 按投放平台汇总 | 对比不同媒体平台消耗和效果 |
| 账户类型 | 账户类型 | 按账户类型汇总 | 对比普通户，绑卡户，代理户等账户表现 |
| 客户 + 投放平台 | 商户ID，客户名称，投放平台 | 按客户+投放平台汇总 | 查看不同客户在各平台的消耗分布 |
| 客户 + 广告账户 | 商户ID，客户名称，广告账户ID，广告账户名称 | 按客户+广告账户汇总 | 查看客户下不同账户表现 |
| 客户 + Campaign | 商户ID，客户名称，Campaign ID，Campaign名称，Campaign状态，推广目标 | 按客户+Campaign汇总 | 查看客户下不同 Campaign 表现 |
| Campaign | Campaign ID，Campaign名称，Campaign状态，推广目标 | 按 Campaign 汇总 | 查看 Campaign 整体表现，不展示客户信息 |
| 国家 | 国家 | 按国家汇总 | 查看不同国家消耗分布 |
| 行业 | 行业 | 按行业汇总 | 查看不同行业消耗表现 |
| 行业 + 国家 | 行业，国家 | 按行业+国家汇总 | 查看行业在不同国家的表现 |
| Campaign + Adset + Ad | Campaign ID，Campaign名称，Campaign状态，推广目标，Adset ID，Adset名称，Adset状态，竞价策略，Ad ID，Ad名称，Ad状态，标题，描述，行动按钮，URL，素材预览 | 按 Campaign+Adset+Ad 汇总 | 分析具体广告效果和素材内容 |
| 日期 + 国家 + 行业 | 日期，国家，行业 | 按日期+国家+行业汇总 | 查看不同日期下国家和行业表现 |

### 7.2 客户字段展示说明

| 场景 | 是否展示商户ID | 是否展示客户名称 | 说明 |
| --- | --- | --- | --- |
| 勾选客户维度 | 是 | 是 | 每一行明确归属于一个客户 |
| 未勾选客户维度，但筛选了单个客户 | 否 | 否 | 筛选条件只限定数据范围，不改变列表维度字段 |
| 未勾选客户维度，且筛选了多个客户 | 否 | 否 | 一行可能包含多个客户数据，展示单个客户信息会误导 |
| 勾选 Campaign 但未勾选客户 | 否 | 否 | 列表按 Campaign 汇总，不展示客户信息 |
| 勾选客户 + Campaign | 是 | 是 | 每行按客户+Campaign分组，可明确归属客户 |

补充说明:

- 如果业务需要知道某一行汇总了哪些客户，V1.0 不在列表中直接展开.
- 后续可扩展 `客户数` 字段或 `客户明细弹窗`，用于查看该行包含的客户范围.
- V1.0 规则保持简单明确: 未勾选客户维度，不展示客户字段.

| 场景 | 是否展示Ad ID | 是否展示Ad名称 | 是否展示Ad状态 | 是否展示标题/描述/行动按钮 | 是否展示URL | 是否展示素材预览 | 说明 |
| --- | --- | --- | --- | --- | --- | --- | --- |

## 8. 页面交互说明

### 8.1 查询交互

| 交互 | 说明 |
| --- | --- |
| 进入页面 | 默认时间范围为最近7天，默认勾选客户维度 |
| 修改筛选项 | 不自动刷新，点击查询后生效 |
| 修改自定义维度 | 不自动刷新，点击查询后生效 |
| 点击查询 | 校验时间范围和维度选择，校验通过后刷新列表 |
| 点击重置 | 恢复默认时间范围，默认筛选和默认维度 |
| 无数据 | 展示空状态，文案为 `暂无数据，请调整筛选条件后重试` |
| 参数错误 | 展示错误提示，例如 `请至少选择一个维度` |

### 8.2 维度选择交互

| 交互 | 说明 |
| --- | --- |
| 勾选维度 | 该维度加入分组字段，列表展示对应维度字段 |
| 取消勾选维度 | 该维度从分组字段移除，列表隐藏对应维度字段 |

## 9. 数据口径

### 9.1 数据来源

| 数据 | 来源 | 说明 |
| --- | --- | --- |
| 广告消耗明细 | Ad层日明细 | 报表聚合的基础事实数据 |
| 客户信息 | 客户主数据 | 提供商户ID，客户名称，行业 |
| 账户信息 | 广告账户主数据 | 提供广告账户ID，广告账户名称，投放平台，账户类型 |
| Campaign信息 | 广告明细或Campaign主数据 | 提供 Campaign ID，Campaign名称，Campaign状态，推广目标 |
| Adset信息 | 广告明细或Adset主数据 | 提供 Adset ID，Adset名称，Adset状态，竞价策略 |
| Ad信息 | 广告明细或Ad主数据 | 提供 Ad ID，Ad名称，Ad状态，标题，描述，行动按钮，URL，素材预览 |

### 9.2 聚合口径

| 类型 | 口径 |
| --- | --- |
| 分组口径 | 按用户勾选的维度字段进行 group by |
| 可加指标 | 直接 sum |
| 比率指标 | 使用汇总后的分子和分母重新计算 |
| 单价指标 | 使用汇总后的消耗和行为数重新计算 |
| 名称字段 | 名称仅展示，不作为唯一聚合依据 |
| ID字段 | Campaign，Adset，Ad 等对象必须使用 ID 作为聚合主键 |

### 9.3 典型指标公式

| 指标 | 公式 |
| --- | --- |
| 消耗 | `sum(spend)` |
| 展示 | `sum(impressions)` |
| 点击 | `sum(clicks)` |
| 转化 | `sum(conversions)` |
| 收入 | `sum(revenue)` |
| CTR | `sum(clicks) / sum(impressions)` |
| CPC | `sum(spend) / sum(clicks)` |
| CPM | `sum(spend) / sum(impressions) * 1000` |
| CPA | `sum(spend) / sum(conversions)` |
| CVR | `sum(conversions) / sum(clicks)` |
| ROAS | `sum(revenue) / sum(spend)` |
