## JavaScript模板示例

### 成功消息模板
```javascript
// 中文
showNotification(`批量充值申请已提交！共${selectedData.length}个账户，总金额$${totalAmount.toFixed(2)}`, 'success');

// 英文
showNotification(`Batch recharge application submitted! ${selectedData.length} accounts, total amount $${totalAmount.toFixed(2)}`, 'success');
```

### 错误消息模板
```javascript
// 中文
showNotification('请至少选择一个账户并设置有效的充值金额', 'warning');

// 英文
showNotification('Please select at least one account and set valid recharge amount', 'warning');
```

## 实施建议

1. **语言切换按钮**：在页面右上角添加语言切换按钮
2. **本地存储**：使用localStorage保存用户语言偏好
3. **动态加载**：页面加载时根据用户偏好加载对应语言
4. **实时切换**：支持页面内实时切换语言，无需刷新页面
5. **默认语言**：默认使用中文简体，支持自动检测浏览器语言
